# DATA501_FinalProject
The final repository for our DATA 501 capstone project.
The original data we collected from different financial terminals is seen in the Excel file named FINAL ESG DATASET_OG. Our data was cleaned using various Excel functions, and the specific data cleaning activities are outlined in our final report.
The cleaned dataset that we use in our analysis is in a CSV file called FINAL_CLEANED_DATASET_V2.
There are four different RMD files in this repository. Three of these files show our analysis for research question 1 (DATA501), research question 2 (DATA501_RQ2), and research question three (DATA501_RQ3). 
The fourth RMD file, DATA501_AlternativeAnalysis, contains an additional analysis we did to each of the three research questions after getting feedback that we should use a method besides linear regression. To do this, we ran a panel quantile regression for each question
